# Portfolio - React Project

This is a React version of your portfolio website built with Vite.

## Project Structure

```
src/
├── components/
│   ├── Hero.jsx           # Main hero section component
│   ├── CardContainer.jsx  # Container for cards
│   ├── ProfileCard.jsx    # Profile card with image
│   ├── WhiteCard.jsx      # White card component
│   └── TextArea.jsx       # Text content and links
├── styles/
│   ├── Hero.css
│   ├── CardContainer.css
│   ├── ProfileCard.css
│   ├── WhiteCard.css
│   └── TextArea.css
├── App.jsx
├── App.css
├── index.css
└── main.jsx
```

## Setup & Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- **Component-based architecture**: Modular React components for easy maintenance
- **Vite bundler**: Fast development and build process
- **Scroll animations**: Fade-in effects on scroll
- **Responsive design**: Mobile-friendly layout
- **Modern styling**: CSS modules for component-specific styles

## Configuration Files

- `package.json` - Project dependencies and scripts
- `vite.config.js` - Vite configuration
- `.gitignore` - Git ignore rules
- `.prettierrc` - Code formatting rules
